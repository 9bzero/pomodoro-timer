# pomodoro-timer

25-minute focus sessions with 5-minute breaks.

The Pomodoro Technique: work for 25 minutes, break for 5, repeat. Every 4 rounds take a longer break (15–30 min).

## Features

- Configurable focus, short break, and long break durations
- Session counter
- Browser notification when time is up
- Soft alarm sound
- Task list — name what you are working on

## Run

```bash
npm install && npm run dev
```
